# Desafio de Turing: Humano ou IA?

Um projeto que simula o Teste de Turing utilizando uma API do Google Gemini.

Desenvolvido em Python e Streamlit

## Como rodar local:

1. Crie um ambiente virtual: 'python -m venv .venv'

2. Ative o ambiente e instale os requisitos: 'pip install -r requirements.txt'

3. Crie uma pasta '.streamlit' com o arquivo 'secrets.toml' contendo sua chave API.

4. Rode: 'streamlit run app.py'
